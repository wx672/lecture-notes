\section{课程说明}
\label{sec-2}

\subsection{课程性质和要求}
\label{sec-2-1}

《Linux操作系统原理与实践》是一门重要的专业基础课。深入理解操作系统的工作原理，了解Linux平
台的软件开发环境，对学生在软件编程、开发方面具有重大意义。 本课程介绍给同学如下内容：
\begin{itemize}
\item 操作系统简介
\item 进程与线程
\item 进程间通信
\item CPU调度
\item 死锁
\item 内存管理
\item 文件系统
\end{itemize}

\subsection{课程重点}
\label{sec-2-2}

\begin{itemize}
\item 进程
\item 内存管理
\item 文件系统
\end{itemize}
