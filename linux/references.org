    大多为英文PDF版。内容涵盖了Linux入门教程、系统管理、网络管理、系统编程、网络编程、应用编程、游戏编程等方方面面。中文书籍较少，且质量不如英文版。因版权原因，这些图书大多不能公开，有需要者，请联系[[mailto:wx672ster@gmail.com][王晓林]]。

* 建议学习步骤
  1. 先熟练掌握命令行的基本操作
  2. 尝试安装。我推荐Ubuntu。
  3. 掌握简单系统管理操作
  4. 了解并适应经典开发环境
     - vim, emacs
     - GCC, GDB, make
     - git
  5. 了解一些可视化开发环境
     - eclipse, netbeans
     - Qt4designer
  6. 了解LAMP开发环境
  7. 了解其它常用工具
     - 系统管理工具
     - 网络管理工具
     - 日常工作工具
  8. 娱乐
